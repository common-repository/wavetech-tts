== Description ==

Convert your Publishes into Audios. listen to them in both admin and front post pages. 
Re-generate audio in case post content changes and enable and disable it for posts. Works on tinyMCE and Gutenberg editors.
 
 * Player that fits almost every theme.
 * Short generation time.
 
== Easy to Use ==

1. Just activate, enable and done! You will see wavetech. link in admin menu and generate button in admin posts pages.
3. Paste and Activate
4. Create new post or edit existing and generate audio.
